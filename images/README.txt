Put your background images here.

Required filenames (the app loads these by name):
  background1.jpg
  background2.jpg
  background3.jpg
  background4.jpg
  background5.jpg

Tips:
- Must be .jpg and named exactly as above (background + number).
- Landscape images look best (e.g. 1920x1080). Lofi / nature / cozy-room style.
- To add MORE than 5: drop in background6.jpg, background7.jpg, ... and then
  open index.html and change the line  const BG_COUNT = 5;  to the new total.
- You can delete this README once your images are in.
